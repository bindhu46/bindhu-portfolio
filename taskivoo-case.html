Personal Project • Progressive Web App

Taskivoo

Taskivoo is a Progressive Web Application (PWA) that helps job seekers organize applications, interviews, recruiter contacts, and follow-ups within a single clean workspace, replacing scattered spreadsheets and notes with an intuitive tracking system.

HTML CSS JavaScript PWA Local Storage

Interactive Demo

The application below is the live version of Taskivoo. Everything you create is stored privately inside your browser using Local Storage.

The Problem

Searching for a job quickly becomes difficult when applications, recruiters, interview dates, salaries, company research, and follow-up reminders are stored across multiple spreadsheets, notes, and emails. As opportunities increase, keeping everything organized becomes stressful and time-consuming.

Taskivoo solves this problem by giving users a single workspace to manage every application from wishlist to offer.

Key Features

Five-Stage Workflow

Every application moves through five clear stages: Wishlist, Applied, Interview, Offer, and Rejected. This gives users a complete picture of their job search.

Dashboard Analytics

Live statistics automatically display the number of applications in each stage, helping users quickly understand their progress.

Powerful Search & Filters

Users can instantly search by company name or filter applications by status to find exactly what they need.

Automatic Follow-up Reminders

Applications that remain in the Applied stage for more than ten days are automatically highlighted, reminding users to contact recruiters.

Dark Mode

A built-in light and dark theme lets users choose their preferred workspace. The selected theme is saved automatically.

Export & Import

Users can export their application data and restore it later, making it easy to move between devices while keeping progress.

Interview Scheduling

Interview dates can be attached to applications and exported as calendar events for quick scheduling.

Progressive Web App (PWA)

Taskivoo can be installed like a native application, allowing quick access from desktop or mobile while providing an app-like experience.

Design Decisions

Rather than creating another dashboard filled with bright colors, Taskivoo was intentionally designed to feel calm and distraction-free. The interface uses warm neutral colors, elegant typography, generous white space, and clear visual hierarchy so users can focus on their job search instead of navigating a complex interface.

Every screen was designed with simplicity in mind. Important actions remain visible while unnecessary visual clutter was removed to create a clean, professional experience.
