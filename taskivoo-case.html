The Problem

Job searching quickly becomes difficult when applications, interview schedules, recruiter conversations, salary notes, company research, and follow-up reminders are scattered across spreadsheets, notebooks, emails, and browser bookmarks.

As opportunities increase, it becomes harder to understand where each application stands or when action is required.

Taskivoo solves this problem by providing one organized workspace where every application moves through a structured hiring pipeline, helping users stay organized throughout their job search.

Key Features

Five-stage workflow. Applications move through Wishlist, Applied, Interview, Offer, and Rejected stages, providing a clear overview of the hiring process.

Dashboard analytics. Live statistics summarize applications by status, helping users monitor progress instantly.

Search and filtering. Quickly locate applications by company name or filter by hiring stage.

Automatic follow-up reminders. Applications that remain in the Applied stage for more than ten days are highlighted, encouraging timely recruiter follow-ups.

Dark mode. Users can switch between light and dark themes, with preferences automatically saved.

Import and export. Application history can be exported and restored across devices.

Interview scheduling. Interview dates can be exported as calendar events for quick scheduling.

Progressive Web App. Taskivoo can be installed on desktop and mobile devices for an app-like experience.

Design Decisions

Instead of creating another dashboard filled with bright colors, Taskivoo was designed around calm typography, generous white space, warm neutral colors, and a distraction-free layout.

Every interface element was intentionally simplified so users can focus on managing applications instead of learning a complicated interface.
