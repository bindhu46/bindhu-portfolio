Taskivoo

A running record of the search — a lightweight, private job application tracker built to replace messy spreadsheets, scattered notes, and forgotten follow-ups with one organized workspace.

Try it

This is the live version of Taskivoo. Everything you add is stored privately in your browser using Local Storage.

The Problem

Searching for a job involves managing far more than a list of applications. Each opportunity comes with company details, job titles, application dates, status updates, recruiter information, salary notes, and important links. As the number of applications grows, keeping everything organized becomes increasingly difficult with spreadsheets or handwritten notes.

Taskivoo addresses this challenge by providing a dedicated application tracking system. Users record each application once and then manage its progress through five clearly defined stages, giving them a centralized, organized view of their entire job search.

What It Does

Five-stage workflow

Organizes every application through five stages—Wishlist, Applied, Interview, Offer, and Rejected. Each entry can be moved between stages with a single click.

Live filters & statistics

Displays a running count of applications by status and lets users instantly filter the board to focus on a single stage.

Automatic follow-up reminders

Applications that remain in the Applied stage for more than 10 days are automatically highlighted, encouraging timely follow-up with recruiters.

Dark mode

Includes a light and dark theme toggle (☾ / ☀) beside the logo. The selected theme is saved automatically and restored on future visits.

Export & import

Allows users to export their application history as text and import it on another device. Duplicate entries are safely merged using unique application IDs.

Interview scheduling

Users can attach interview dates to applications and download a ready-to-use .ics calendar event for quick scheduling.

Progressive Web App (PWA)

Built as a Progressive Web App using a web manifest and service worker, allowing installation on desktop and mobile devices with offline capabilities when self-hosted. The portfolio demonstrates the app inline as part of the case study.
