🚀 Taskivoo

Personal Job Application Tracker

The Problem

Job searching quickly becomes difficult to manage. Information such as company names, job titles, interview dates, recruiter contacts, and application status often ends up scattered across spreadsheets, notes, and emails. Keeping everything organized becomes time-consuming and increases the risk of missing important follow-ups.

Why I Built Taskivoo

I wanted a simple application that allows job seekers to manage every application in one place. Instead of switching between spreadsheets and notes, users can track their complete job search through an intuitive and organized interface.

What It Does

  • Track applications from Wishlist to Offer
  • Add, edit and delete applications
  • Search companies instantly
  • Filter by application status
  • Dashboard showing application statistics
  • Automatic follow-up reminders
  • Dark Mode support
  • Export and Import application data
  • Interview scheduling with calendar (.ics) export
  • Installable as a Progressive Web App (PWA)

Design

Taskivoo follows a clean journal-inspired interface designed to reduce the stress of job searching. The layout focuses on simplicity, readability, and fast navigation while maintaining a modern professional appearance.

Under the Hood

Taskivoo was built using plain HTML, CSS, and JavaScript without external frameworks. Application data is stored locally using browser localStorage, allowing users to manage their information privately without creating an account or relying on a backend server.
