Try it

This is the actual working application running live below. Everything you add is stored privately in your own browser using LocalStorage—nothing is uploaded, shared, or sent to a server.

a running record of the search

The Problem

Managing dozens of job applications across spreadsheets, browser bookmarks, emails, and handwritten notes quickly becomes difficult. Important follow-ups are missed, interview details become scattered, and it is hard to understand overall job search progress.

The Solution

Taskivoo was designed as a lightweight personal job application tracker that keeps every application, interview, follow-up, and note in one organized workspace. The application runs entirely in the browser using Local Storage, giving users a private and fast experience without requiring an account.
