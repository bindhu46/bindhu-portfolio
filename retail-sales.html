📊 Retail Sales Analysis

End-to-end SQL and Power BI analysis of 9,800 retail order line items.

Executive Summary

This project analyzes 9,800 retail sales records using SQL, MySQL, Excel, and Power BI to evaluate sales performance, profitability, customer behavior, and regional trends. A normalized relational database was designed, SQL queries were used to answer key business questions, and interactive dashboards were built to visualize KPIs and business insights.

Problem Statement

Retail businesses often generate large volumes of sales data but struggle to identify which products, regions, and customer segments contribute the most to profitability. This project was designed to transform raw transactional data into meaningful business insights using SQL, MySQL, and Power BI.

Technology Stack

SQL MySQL Power BI Excel Data Analysis Data Visualization
